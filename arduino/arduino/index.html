Utilização da api

Lampada 1
<id> Identificador da sessão
<lampada1> - Ligar/desligar a lampada 1, <ligado=1,desligado=0>
http://simuladorcasa.mooo.com/server/index.php/house/update/id/<id>/lampada1/<ligado=1,desligado=0>

Exemplo:
Ligar a lamapda 1 com identificador 167
http://simuladorcasa.mooo.com/server/index.php/house/update/id/167/lampada1/1
Desligar a lamapda 1 com identificador 167
http://simuladorcasa.mooo.com/server/index.php/house/update/id/167/lampada1/0

Utilizando a temporização das Lâmpadas

<id> Identificador da sessão
http://simuladorcasa.mooo.com/server/index.php/criarTimeout/id/<id>/lampada1tempo/<30>

http://simuladorcasa.mooo.com/server/index.php/criarTimeout/id/167/lampada1tempo/30
No exemplo acima a lamapada1 será desligada depois de 30 segundos.

Para lampada2 utilize lampada2tempo
Para lampada3 utilize lampada3tempo
Para lampada4 utilize lampada4tempo
Para lampada5 utilize lampada5tempo
Para lampada6 utilize lampada6tempo
